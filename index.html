  • CARA MEMBUAT RDP & VPS DI BOT WHATSAPP PEDIAWCH

    RDP
    Format command: #createrdp|negara|size|namardp|sistemoperasi|perpanjangotomatis ya/tidak
    Contoh: #createrdp|singapura|4c8gi|rdpwp|win10|ya

    VPS
    Format command: #createvps|negara|size|namavps|sistemoperasi|perpanjangotomatis ya/tidak
    Contoh: #createvps|singapura|4c8gi|vpswp|ubuntu20|ya

    NOTE
    Jika status perpanjang otomatis kamu pilih "ya". Maka otomatis jika ip RDP/VPS kamu sudah habis, otomatis akan diperpanjang dan memotong saldo kamu. Pastikan saldo cukup sesuai size yang dipilih
    List Negara, Size, Sistem Operasi sudah ada di #rdp dan #vps

  • CARA PERPANJANG RDP & VPS DI BOT WHATSAPP PEDIAWCH

    RDP
    Format command: #perpanjangrdp|iprdpnya|ya/tidak
    Contoh: #perpanjangrdp|128.199.199.199:50123|tidak

    VPS
    Format command: #perpanjangvps|ipvpsnya|ya/tidak
    Contoh: #perpanjangvps|128.199.199.199|tidak

    NOTE RDP & VPS
    Fungsi ini hanya untuk merubah status ip kamu mau diperpanjang atau tidak.
    Jika sebelumnya ip kamu statusnya "ya", maka tinggal kamu rubah menjadi "tidak". Maka otomatis ip kamu akan berubah status perpanjangnya. bisa di cek di #listakun untuk melihat statusnya sudah berubah atau belum

  • CARA RECREATE RDP & VPS DI BOT WHATSAPP PEDIAWCH

    RDP
    Format command: #recreaterdp|iprdpnya|sistemoperasi
    Contoh: #recreaterdp|128.199.199.199:50123|win10

    VPS
    Format command: #recreatevps|ipvpsnya|sistemoperasi
    Contoh: #recreatevps|128.199.199.199|ubuntu24

    NOTE RDP & VPS
    Fungsi ini untuk mengganti ip rdp atau vps kamu dengan ip dan sistem operasi yang baru. Misal sebelumnya 128.199.199.199:50123 maka akan terganti 134.199.199.128:50123. Dan misal sebelumnya win10 mau diganti ke win2022 bisa
    Fungsi ini dikenakan biaya Rp.1000 setiap kali kamu melakukan recreate pada RDP & VPS. Karena untuk menghindari spam pengulangan pembuatan secara terus menerus
    Pastikan lakukan recreate RDP & VPS sewajarnya saja, jika terdeteksi melakukan spam. Akan saya banned data akun di BOT dan saldo hangus

  • CARA START, STOP, RESTART RDP & VPS DI BOT WHATSAPP PEDIAWCH

    START RDP
    Format command start: #startrdp|iprdpnya
    Contoh: #startrdp|128.199.199.199:50123

    STOP RDP
    Format command stop: #stoprdp|iprdpnya
    Contoh: #stoprdp|128.199.199.199:50123

    RESTART RDP
    Format command restart: #restartrdp|iprdpnya
    Contoh: #restartrdp|128.199.199.199:50123

    START VPS
    Format command start: #startvps|iprdpnya
    Contoh: #startvps|128.199.199.199

    STOP VPS
    Format command stop: #stopvps|iprdpnya
    Contoh: #stopvps|128.199.199.199

    RESTART VPS
    Format command restart: #restartvps|iprdpnya
    Contoh: #restartvps|128.199.199.199

    NOTE
    Fungsi ini untuk melakukan start, stop dan restart pada RDP & VPS kamu. Misal rdp atau vps kamu tidak sengaja di shutdown, kamu bisa menyalakan kembali dengan fungsi #startrdp atau #startvps
    Pastikan setelah menjalankan fungsi fungsi diatas, tunggu sekitar 10 sampai 15 menit, lalu loginkan kembali. jika lebih dari 30 menit tidak bisa di akses, bisa kontak admin di #contact

  • CARA REBUILD RDP & VPS DI BOT WHATSAPP PEDIAWCH

    RDP
    Format command: #rebuildrdp|iprdpnya|sistemoperasi
    Contoh: #rebuildrdp|128.199.199.199:50123|win2022

    VPS
    Format command: #rebuildvps|ipvpsnya|sistemoperasi
    Contoh: #rebuildvps|128.199.199.199|ubuntu22

    NOTE
    Fungsi ini untuk mengganti sistem operasi RDP dan VPS kamu. Misal pada RDP kamu tadinya windows 10, mau diganti ke windows 2022. begitu juga untuk VPS misal mau diganti ke ubuntu 22
    Fungsi ini tidak dikenakan biaya, pastikan lakukan rebuild dengan sewajarnya tanpa melakukan spam ya
